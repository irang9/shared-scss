Breakpoints

반응형 디자인을 위한 breakpoint 값들

Breakpoint Map

이름 값 용도 사용 예시
xxs 320px 초소형 모바일 @include up("xxs")
xs 360px 소형 모바일 @include up("xs")
sm 480px 중형 모바일 @include up("sm")
md 576px 대형 모바일 @include up("md")
lg 768px 태블릿 @include up("lg")
xl 992px 소형 데스크톱 @include up("xl")
2xl 1024px 데스크톱 @include up("2xl")
3xl 1200px 대형 데스크톱 @include up("3xl")
4xl 1400px 초대형 데스크톱 @include up("4xl")

Core Mixins

Map 기반 breakpoint mixin 사용법:

Mixin 설명 사용 예시 적용 범위 예시
@include up("xs") 위로 (min-width) - Mobile First @include up("xs") { ... } 360px 이상 (소형 모바일 이상)
@include down("md") 아래로 (max-width) - Desktop First @include down("md") { ... } 575.98px 이하 (대형 모바일 이하)
@include between("xs", "lg") 범위 (between) - 특정 범위 지정 @include between("xs", "lg") { ... } 360px ~ 767.98px (소형 모바일 ~ 태블릿)

사용 예시

.container {
    padding: 16px;
    
    @include up("md") {
        padding: 24px;  // 576px 이상에서 적용
    }
    
    @include down("sm") {
        padding: 12px;  // 479.98px 이하에서 적용
    }
    
    @include between("md", "xl") {
        max-width: 1200px;  // 576px ~ 991.98px 범위에서 적용
    }
}

기존 변수 (하위 호환성)

다음 변수들은 Map 기반 breakpoint에서 자동으로 생성됩니다:

  • $mobile-xxs = $bp["xxs"]
  • $mobile-xs = $bp["xs"]
  • $mobile-sm = $bp["sm"]
  • $mobile = $bp["md"]
  • $desktop-xs = $bp["lg"]
  • $desktop-sm = $bp["xl"]
  • $desktop = $bp["2xl"]
  • $desktop-lg = $bp["3xl"]
  • $desktop-xl = $bp["4xl"]
