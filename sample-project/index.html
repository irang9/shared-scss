RexBox 사용 예제

프로젝트별 커스터마이징과 오버라이드 방법을 보여주는 샘플 프로젝트입니다.

시작하기

Features

RexBox의 변수, mixins, utilities를 활용한 컴포넌트 예제입니다.

palette

커스텀 색상

_config.scss에서 primary, secondary 색상을 오버라이드하여 프로젝트 브랜드 색상을 적용할 수 있습니다.

code

Mixins 활용

rounded, transition 등 RexBox의 mixins를 사용하여 일관된 스타일을 적용할 수 있습니다.

devices

반응형 디자인

breakpoint mixins를 사용하여 모바일, 태블릿, 데스크톱에 최적화된 레이아웃을 구현할 수 있습니다.
